NexWear is a full-stack Fashion E-Commerce Web Application developed using Python Flask, MySQL, HTML, CSS, and Bootstrap.

Features:
• User Registration and Login
• Admin Dashboard
• Category Management
• Product Management
• Shopping Cart
• Order Management
• User Profile Management
• Session Handling
• Responsive Design
• MySQL Database Integration

This project demonstrates full-stack web development concepts including CRUD operations, authentication, database connectivity, session management, and e-commerce workflow implementation.
